# Project Setup and Launch Guide

## Prerequisites
- Python 3 installed on your system
- Project files downloaded/migrated to your local machine

## Setup Instructions

1. Clone or download the project files to your local machine.
2. Navigate to the project directory in your terminal or file explorer.
3. Run the `setup.bat` file by double-clicking it or executing it from the command line.
   - This script will create a virtual environment
   - Install all the required dependencies
   - Activate the virtual environment

## Launching the Application

1. After completing the setup, run the `launch.bat` file by double-clicking it or executing it from the command line.
   - This script will activate the virtual environment
   - Launch the Streamlit application
   - The application should open automatically in your default web browser

## File Descriptions

- `setup.bat`: Creates and activates the Python virtual environment, installs required packages
- `launch.bat`: Activates the virtual environment and starts the Streamlit application
- `app.py`: Main application code

## Troubleshooting

If you encounter any issues during setup or launch:
- Ensure Python 3 is correctly installed and added to your system PATH
- Verify all project files have been properly migrated
- Check console output for specific error messages
